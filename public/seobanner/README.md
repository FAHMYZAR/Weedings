# Banner SEO

Letakkan file banner.png di folder ini dengan ukuran 1200x630 pixels untuk optimal social media sharing.

File ini akan muncul ketika URL undangan di-share ke:
- Facebook
- WhatsApp  
- Twitter
- Instagram
- LinkedIn
- Dan platform social media lainnya

Format yang disarankan:
- Ukuran: 1200x630 pixels
- Format: PNG atau JPG
- Nama file: banner.png
